### 功能说明
用于线下购物时临时计算一下自己的商品，就用来在有优惠券的时候凑单用

### 最后更新 

2024.12.04

### 使用方法

```bash

git clone https://github.com/afyi/cart.git

cd cart

yarn

yarn dev 

```
